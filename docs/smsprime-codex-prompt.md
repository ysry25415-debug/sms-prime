# SMSPrime Codex Prompt

Build an original premium SaaS platform inspired by the workflow clarity of modern SMS verification dashboards, without copying any existing brand, layout, or visual identity.

The interface should feel familiar to users of SMS verification platforms:

- Fast service search
- Clear balance display
- Country and service tables
- Purchase flow with minimal clicks
- Countdown timers for active orders
- One-click copy for numbers and codes
- Strong status indicators
- Clean pricing cards
- Reliable provider status panels

Important visual direction:

- Use our brand colors only
- Dark-first interface
- Background `#0F172A`
- Surface `#1E293B`
- Borders `#334155`
- Primary `#3B82F6`
- Success `#22C55E`
- Warning `#F59E0B`
- Danger `#EF4444`
- Text `#F8FAFC`
- Secondary text `#94A3B8`

Design principles:

- Premium SaaS quality
- Clean spacing
- Rounded corners around 16px
- Glassmorphism panels
- Soft gradients
- Strong typography
- Smooth motion
- Accessible interactions

Layout expectations:

- Sticky blurred header
- Left sidebar in dashboard pages
- Hero section with search and CTA
- Statistics cards
- Service table
- Activity feed
- Provider/API status cards

Do not mimic any competitor visually. Keep the workflow familiar but make the branding and aesthetic fully original to SMSPrime.
